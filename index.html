📚 Complete User Guide

🔑 Step 1: Get Your API Key

  1. Go to console.anthropic.com
  2. Sign up (free account, $5/month credits)
  3. Click "API Keys" in the left sidebar
  4. Click "Create Key"
  5. Copy the key (looks like: sk-ant-...)
  6. Keep it private!

📊 Feature 1: Compare URLs

What it does: Analyzes your content vs competitors to find ranking opportunities.

  1. Paste your URL (e.g., yoursite.com/page)
  2. Paste 1-3 competitor URLs (one per line)
  3. Click "Analyze"
  4. Paste your API key when prompted
  5. Get E-E-A-T scores, gaps, and recommendations

✨ Feature 2: Rewrite Content

What it does: Optimizes your existing content for better rankings.

  1. Enter your target keyword
  2. Select page type (Blog, Service, Homepage, Case Study)
  3. Paste your current content
  4. Click "Rewrite"
  5. Get optimized version ready to copy

🚀 Feature 3: Generate Content

What it does: Creates new, high-quality content from scratch.

  1. Enter primary keyword
  2. Add secondary keywords (optional)
  3. Select page type and target audience
  4. Describe your company/brand
  5. Click "Generate"
  6. Get complete, ready-to-publish content

🐛 Troubleshooting

Getting "Failed to fetch"?

  • ✅ Make sure API key is correct (starts with sk-ant-)
  • ✅ Check you have API credits at console.anthropic.com
  • ✅ Try refreshing the page
  • ✅ Clear browser cache (Ctrl+Shift+Delete)
  • ✅ Try a different browser
  • ✅ Press F12 to open Developer Console for detailed errors

Getting "API Error"?

  • ✅ Your API key format is wrong
  • ✅ Your API key has expired
  • ✅ You've run out of API credits
  • ✅ Create a new key at console.anthropic.com
