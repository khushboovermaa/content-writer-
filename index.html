📚 Complete User Guide

Overview

This all-in-one tool helps you create, optimize, and analyze content using Google's E-E-A-T framework.

📊 Feature 1: Compare URLs

Purpose: Analyze your content vs competitors to find ranking opportunities.

  1. Enter your URL
  2. Enter 1-3 competitor URLs (one per line)
  3. Click "Analyze"
  4. Get E-E-A-T scores, gaps, and recommendations

✨ Feature 2: Rewrite Content

Purpose: Optimize existing content for better rankings.

  1. Enter your target keyword
  2. Select page type
  3. Paste your content
  4. Click "Rewrite"

🚀 Feature 3: Generate Content

Purpose: Create new, high-quality content from scratch.

  1. Enter primary keyword
  2. Add secondary keywords (optional)
  3. Select page type and audience
  4. Click "Generate"

🔑 API Key Setup

  1. Go to console.anthropic.com
  2. Sign up (free tier available)
  3. Create an API key
  4. Paste it when the tool asks
